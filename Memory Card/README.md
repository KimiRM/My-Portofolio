# Memory Card App 📝

A beautiful and interactive memory card management application that allows users to create, edit, view, and delete digital cards with titles and descriptions.

## ✨ Features

- ➕ **Create Cards** - Add new memory cards with title and description
- ✏️ **Edit Cards** - Modify existing card content
- 👁️ **View Details** - See complete card information
- 🗑️ **Delete Cards** - Select and delete multiple cards
- 🎨 **Modern UI** - Dark theme with smooth animations
- 📱 **Responsive Design** - Works on different screen sizes

## 🛠️ Technologies Used

- HTML5
- CSS3 (Flexbox, Grid)
- Vanilla JavaScript (ES6+)
- DOM Manipulation

## 🚀 How to Use

1. Click **"Save New Item"** to create a card
2. Fill in the title and description
3. Click **"Show All Items"** to view your cards
4. Use **"Edit"** button to modify a card
5. Click **"View Details"** to see full content
6. Select multiple cards and click **"Delete Item"** to remove them
